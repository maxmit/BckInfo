BckupNix Bug Report


Submitting a bug Report:

Answer the following questions and include in your bug report::


Do you have Python installed (you need Python installed to run this app), most if not all nix* OS's have python installed, but asking anyway?

What version of Python are you using (example -> 2 or 3, you need version 3)?

Access a command-line/terminal window:

  • Linux: Ctrl-Alt-T, Ctrl-Alt-F2
  • MacOS: Finder > Applications > Utilities > Terminal
  • Windows: keyboard buttons -> windows key+r, then type "cmd"

    • Run command -
    python3 --version

    Also answer the following::

    What is the name and version of your OS (Operating System)?

    Are you running this program in a virtual machine, or hardware machine?

    Can you duplication the issue?

    Can you send the actual error message?


    Submit all the information above to Max:
    max@ofoperations.com
