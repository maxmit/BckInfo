Vision:

The vision of this project is to build, modify, and use a simple tool to backup and protect computer systems (software) without any difficult or complex use of the tool (application).

I want this backup tool to not only be simple to use, but simple to build and to implement into any computer operating system, making it cross platform compatible and portable into multiple environments.


Mission:

The mission of this community and the project will be to use open source tools, documentation, and programs (Python3 in this case), to create a application that is beneficial and helpful to everyone who is interested in protecting and keeping their systems safe from harms way.


Values:

I want to make this project and the community that supports it, VERY easy to join, contribute to, and collaborate with, whether you can code or not.

Everyone, no matter what our differences are, should feel free to join this community, and input your talents to helping not only the project but all of us the world over.
